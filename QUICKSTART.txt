🚀 БЫСТРЫЙ СТАРТ
================

📦 1. УСТАНОВКА ЗАВИСИМОСТЕЙ
-----------------------------
Откройте PowerShell/CMD в этой папке и выполните:

    pip install -r requirements.txt


🎯 2. ЗАПУСК (3 ВАРИАНТА)
-----------------------------

Вариант А - GUI Launcher (Рекомендуется):
    python launcher.py

Вариант Б - Прямой запуск сервера:
    python server.py

Вариант В - Собрать в .exe:
    python build.py
    или просто запустите build_simple.bat


📱 3. ДОСТУП К СЕРВЕРУ
-----------------------------
После запуска откройте в браузере:
    http://192.168.X.X:8000  (IP будет показан при запуске)

Или отсканируйте QR-код с телефона!

🔧 НАСТРОЙКА
-----------------------------
В GUI launcher можно настроить:
- Порт сервера (по умолчанию 8000)
- Папку для загрузок (по умолчанию uploads/)


❓ ЧАСТЫЕ ВОПРОСЫ
-----------------------------
Q: Как получить доступ с телефона?
A: Убедитесь что оба устройства в одной WiFi сети, 
   затем отсканируйте QR-код или введите IP адрес

Q: Не могу собрать в .exe?
A: Убедитесь что установлены все зависимости:
   pip install -r requirements.txt

Q: Сервер не запускается?
A: Проверьте что порт 8000 не занят другой программой
   Можете изменить порт в GUI launcher
