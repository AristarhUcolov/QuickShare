3. Click "Start Server" / "Запустить сервер"

4. Click 🌐 button to switch language
   Нажмите 🌐 для смены языка


FOR DEVELOPERS / ДЛЯ РАЗРАБОТЧИКОВ:
────────────────────────────────────

Run from source:
    python launcher.py
    or
    python server.py

Build .exe files:
    python build_all.py

Test language switching:
    http://localhost:8000?lang=en
    http://localhost:8000?lang=ru

╔════════════════════════════════════════════════════════════════╗
║                  🔧 TECHNICAL DETAILS                          ║
╚════════════════════════════════════════════════════════════════╝

ARCHITECTURE:
- Flask backend
- Jinja2 templates
- Cookie-based language storage
- Modular translation system

FILES STRUCTURE:
translations.py:
  - TRANSLATIONS dict with 'en' and 'ru' keys
  - get_translation(lang) function
  - Easy to add new languages

server.py:
  - get_current_language() - detects from cookie/URL
  - format_size(size, lang) - localized file sizes
  - All routes support multi-language

LANGUAGE DETECTION:
1. Check URL parameter (?lang=en)
2. Check cookie (language)
3. Default to 'en'
